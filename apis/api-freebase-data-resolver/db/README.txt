README:
-------

This folder contains the database of the file.

The database is called "location.h2.db" and it has three tables with location and one to keep results to next queries.

ISO3661_COUNTRY => Country names and the codes

ISO3661_FIPS_10_4_REGION => Region with country codes

ISO3661_FIPS_10_4_CITY => Cities with the region name and its country name

Folders:

	clean => Contains a clean copy.
	test-copy => Contains a copy used for tests and development.
